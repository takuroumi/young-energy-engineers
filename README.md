# About Young Energy Engineers Group / 若手エネルギー技術研究会について
「若手エネルギー技術研究会」は、エネルギー「技術」に関心のある若者が集まり、勉強や議論をし合うためのコミュニティです。気候変動やエネルギー問題そのものは政治的/倫理的観点から議論が多くかわされている中、これらの社会課題の解消に繋がる技術に興味を持ち、実際に研究・技術開発・社会実装の行動を起こすという人は少ないのが現状です。若手エネルギー技術研究会は、これからの電力/エネルギーインフラの構築を担う若手世代向けに、技術を学ぶ/行動を起こす方々が集まり交流ができる場を提供することを目的に設立されました。


# Activity / 活動内容
- Slackグループで技術情報の共有・議論
- 内部向けの技術勉強会の開催

# For Who Young Energy Engineers Group is / 若手エネルギー技術研究会は誰のためのコミュニティなのか
「若手エネルギー技術研究会」は、以下の気概を持った方のために用意されたコミュニティです。
- エネルギー技術に関心があり、技術を学ぶために情報収集・情報交換したい方。同じ業界で研鑽している方々と切磋琢磨したい方。
- 気候変動やエネルギー問題に対して政治的・倫理的な観点で議論するだけでなく、客観的に物事・課題を判断出来るレベルの技術知識を身につけたい人
- 仕事や就活など，個人の場で応用・活用できる実践的な技術知識を身につけたい方
- 「若手」にあたる方

# Admission / 入会
- 「若手エネルギー技術研究会」は招待制のコミュニティです。Slackグループへの参加を以て入会とみなし、Slackグループからの脱退を以て退会とみなします。
- 入会する前には、必ず「行動規範」を読んで内容に同意してください。入会後「行動規範」に沿わない行動が散見され、運営からの注意を受けたあとも反省の意が見られなかった場合、「若手エネルギー技術研究会」からの強制退会の対応をとらせていただく場合がございます。
- 入会を希望する方は、事前にメンバーへ入会希望の旨をお声掛けください。
- 「若手エネルギー技術研究会」の入会および退会には費用はかかりません。

# Code of Cunduct / 行動規範
「若手エネルギー技術研究会」コミュニティでは以下の行動規範を採用し、参加するすべての人に安全で前向きなコミュニティ体験を提供するため、これらの参加ガイドラインに同意することを求めています。

これらのガイドラインは、すべての人々が安全に本コミュニティに参加し、活躍いただけることを目的としています。特に以下の如何に関わらず、すべての人が健全に過ごせるようなガイドとしてご利用ください。

- バックグラウンド
- 家族のステータス
- ジェンダー
- アイデンティティ
- 配偶者の有無
- 性別
- 性的志向
- 母国語
- 年齢
- 能力
- 人種およびまたは民族
- 出身国
- 社会経済的地位
- 宗教
- 地理的位置
- 多様性のそのほかの側面

本行動規範では、本コミュニティ参加者に期待される行動と許容されない行動の両方について概説します。

## 期待される行動
次の行動は、すべての関係者に期待されています。

- 思いやり
- 公平さ
- 透明性
  - できるだけ個人間のDMは使用しないでください。個人間のDMが使用され、DMで何かを決定した場合、他の人は何かが起こる理由を理解できません。
- さまざまな視点/異なる立場への理解、リスペクト
- 確信が持てないときに助けを求める

## 許容されない行動
次の行動は、受け入れられないものとみなされます。

- 暴力
- 個人攻撃
- 軽蔑的な言葉
- 望ましくない性的な注目または身体的接触
- 破壊的な行動
- 過度な要求
- その他容認できない行動

## 注意される発言
次のような発言は、特定個人に対してのものであれ、不特定多数に対するものであれ、本コミュニティでは受け入れられないものとみなします。たとえ褒め言葉を意図した発言であってもハラスメントにあたりえる場合は注意の対象となります。

- セクシズムやレイシズム、その他のバイアスに基づく発言
- ルッキズム、その他のバイアスに基づく発言
- マンスプレイニングやトーンポリシング、その他のバイアスに基づく発言
  - マンスプレイニング（主に男性から女性に行われる上から目線で説明）や過度なトーンポリシング（話し方や態度の批判）はバイアスを示す行動として、注意の対象となります。
- 国籍に関するバイアスに基づく発言
- エイジズム、その他のバイアスに基づく発言
  - 若手エネルギー技術研究会の「若手」は特定の年齢を揶揄する意は含みません。本コミュニティに置いても年齢に関する不必要な発言は歓迎しません。

## 運営者の責任
運営は以下の責任を持ちます。

- 本コミュニティが公平であるよう努めること
- 本コミュニティが率直であるよう努めること
- 本コミュニティが透明であるよう努めること
- 本コミュニティの目的の説明に努めること

## 報告
上記のような容認されない/容認できない行動を経験していると思われる場合は、所定のフォームから連絡してください。
あなたの状況の簡潔な説明を受けた後、運営は次のステップを検討し決定します。

また、危険な状況、苦痛に直面している人、またはこれらのガイドラインに違反している状況を観察した場合は、その状況があなたに起こっていない場合でも、報告してください。

これらのガイドラインに違反していると不当に非難されていると思われる場合は、同じ報告プロセスに従ってください。

## 容認できない行動への対処
意思決定の権限を持つ者を含め、本コミュニティからの悪い行動は容認されません。本コミュニティの活動から人々を除外する意図的な努力（ガイドラインまたはその他の運営アクションの結果の一部を除く）は受け入れられず、適切に対処されます。

嫌がらせ/差別の報告は、スペース、イベント、またはアクティビティの安全性に責任を持つ人々によって迅速かつ徹底的に調査されます。状況に対処するために適切な措置が取られます。

容認できない行動を止めるように求められた人は誰でもすぐに遵守することが期待されます。これらのガイドラインに違反すると、一時的またはイベント期間中、イベントまたはオンラインスペースを離れるように求められたり、スペースへの参加が禁止されたり、将来のイベントや活動が永久に禁止されることがあります。

さらに、報告プロセスを悪用した参加者は、これらのガイドラインに違反しているとみなされ、同じ結果になります。特に報復または除外するための虚偽の報告は受け入れられません。

## 行動規範の改定
「若手エネルギー技術研究会」運営は本コミュニティのより良い運営を目指す目的で、本行動規範の内容を変更する場合があります。本行動規範の内容の変更（誤字脱字の修正のような軽微な変更を除く）には、「若手エネルギー技術研究会」運営の承認を要します。内容変更のリクエストの際はGitHubにissueを作成いただくか、Slackグループ上に投稿ください。
